Nettsiden deles opp i seksjoner som alle har sitt eget HTML document, med et tilhørende CSS document. Alle seksjonene kan settes sammen i hovedsiden med JS/PHP eller kompileres sammen før nettsiden publiseres. Dette lar oss bruke github til å jobbe på hovedsiden samtidig. Stiler som deles mellom flere eller alle seksjoner lagres i ett CSS document som også importere alle CSS filene.
